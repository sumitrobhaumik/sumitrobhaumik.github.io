But Why Terminal Apps?

Pros

  1. Extremely fast startup
  2. Low resource usage
  3. Less dependancies -> More stable
  4. Remote access is very easy as long as you have ssh
  5. Community driven, very hackable

Cons

  1. Esoteric design - There is a slight learning curve.
  2. Mostly keyboard-driven
  3. Hard to access through mobile
  4. Hope your terminal app is still actively developed

My choice of terminal apps

  • htop - System monitor
  • ranger - File manager
  • vim - Editor
  • screen - Terminal multiplexer
  • cmus - Music player
  • tremc - curses based torrent manager to be used in conjunction with transmission (LINK)
  • vis - Music visualizer (LINK)
  • speedometer - Network bandwidth monitor
  • tty-clock - A clock

All these apps should be availble via your package manager, other than the ones for which I have given a Github link.

Arch users, go to AUR (obviously)
