But Why Terminal Apps?

Ever since I switched to i3, I have simply become fascinated by terminal apps. I've listed all the reasons why I think they are cool.

Pros

  1. Extremely fast startup
  2. Low resource usage
  3. Less dependancies -> More stable
  4. Remote access is very easy as long as you have ssh
  5. Community driven, very hackable
  6. [BONUS] They make you look like a hacker /s

But all is not perfect

Cons

  1. Esoteric design - There is a slight learning curve, and sometimes very high.
  2. Mostly keyboard-driven
  3. Hard to access through mobile
  4. Hope your terminal app is still actively developed
  5. A lot of time needs to be invested to tune it to your liking, but it's a one time effort

My choice of terminal apps

  • htop - System monitor
  • ranger - File manager
  • vim - Editor
  • screen - Terminal multiplexer
  • cmus - Music player
  • tremc - curses based torrent manager to be used in conjunction with transmission (LINK)
  • vis - Music visualizer (LINK)
  • speedometer - Network bandwidth monitor
  • tty-clock - A clock

All these apps should be availble via your package manager. For apps which might not be available in common repos, I have given a Github link.
